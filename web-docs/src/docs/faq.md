**O gateway armazena dados de cartão?**
Não. Para cartão, usamos provedores terceirizados, recebendo apenas tokens.

**PIX funciona com quais bancos?**
A arquitetura suporta múltiplos PSPs; inicialmente, integrações com bancos e provedores com API de PIX Cobrança.

**Existe ambiente sandbox?**
Sim. Use a URL de sandbox e chaves de teste para simular pagamentos sem valor real.
