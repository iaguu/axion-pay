### Passo a passo de integração

1. Solicite sua chave de API no painel do gateway.
2. Configure as URLs de Sandbox e Produção em variáveis de ambiente.
3. Implemente criação de cobrança PIX (início da jornada).
4. Implemente tratamento de webhooks.
5. (Opcional) Adicione fluxo de cartão via PSP integrado ao gateway.
6. Teste cenários de sucesso, erro, expiração e cancelamento.
